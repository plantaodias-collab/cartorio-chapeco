🔧 Conectar ao banco de dados

Esta configuração é feita uma única vez por computador. O banco é gratuito.

Passo 1 — Criar projeto Firebase

Acesse console.firebase.google.com → "Adicionar projeto" → nome: cartorio-chapeco → desative Google Analytics → "Criar projeto".

Passo 2 — Criar Realtime Database

Menu lateral: "Criar" → "Realtime Database" → "Criar banco de dados" → localização: us-central1 → "Iniciar no modo de teste" → "Ativar".

Passo 3 — Copiar a URL e colar abaixo

A URL aparece no topo após criar. Termina com .firebaseio.com.

Exemplo: https://cartorio-chapeco-default-rtdb.firebaseio.com

❌ URL inválida. Deve começar com https:// e conter firebaseio.com
Opcional - Ativar login

Cole aqui a configuracao do app Web do Firebase para habilitar login por e-mail e senha. Se ficar vazio, o painel continua abrindo como hoje.
