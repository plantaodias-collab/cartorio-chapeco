🔧 Configurar Banco de Dados

Esta configuração é feita uma única vez. Siga os passos abaixo para criar o banco de dados gratuito e colar a URL aqui.

Passo 1 — Criar projeto Firebase

Acesse console.firebase.google.com → clique em "Adicionar projeto" → dê um nome (ex: cartorio-chapeco) → desative o Google Analytics → clique em "Criar projeto".

Passo 2 — Criar o Realtime Database

No menu lateral clique em "Criar" → "Realtime Database" → clique em "Criar banco de dados" → escolha localização us-central1 → selecione "Iniciar no modo de teste" → clique em "Ativar".

Passo 3 — Copiar a URL do banco

Após criar, aparece a URL do banco. Ela termina com .firebaseio.com. Exemplo:
https://cartorio-chapeco-default-rtdb.firebaseio.com
